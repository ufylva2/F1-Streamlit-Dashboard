# F1 Analytics Hub

Este projeto foi desenvolvido no âmbito da minha formação em **Data & Business Analysis** com o objetivo de explorar dados históricos da Fórmula 1 através de um dashboard interativo criado em **Python** e **Streamlit**.

Ao longo do desenvolvimento procurei aplicar boas práticas de análise de dados, organização de código e visualização de informação, criando uma aplicação que permite explorar estatísticas de pilotos, equipas, circuitos e corridas de forma simples e intuitiva.

---

# Funcionalidades

O dashboard inclui várias páginas de análise, entre elas:

- Visão geral da Fórmula 1
- Pilotos com mais corridas
- Nacionalidades com mais vitórias
- Análise de Pit Stops
- Evolução do número de corridas por época
- Circuitos com mais Grandes Prémios
- Equipas com mais derrotas
- Comparação entre qualificação e resultado final
- Fiabilidade e abandonos por equipa

Cada página permite explorar diferentes indicadores através de gráficos e métricas interativas.

#  Como executar

1. Clonar o repositório

```bash
git clone https://github.com/ufylva2/F1-Streamlit-Dashboard.git
```

2. Instalar as dependências

```bash
pip install -r requirements.txt
```

3. Executar a aplicação

```bash
streamlit run "🏎️Fórmula_1.py"
```
